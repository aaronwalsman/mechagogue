# MaxNIST Examples

Training scripts for multi-digit classification on [MaxNIST dataset](../../data/maxnist.py).

- `train_maxnist.py` - Standard supervised learning with backpropagation (AdamW)
- `train_maxnist_ga.py` - Genetic algorithm training with various architectures
- `train_mutable_maxnist_ga.py` - GA training with dynamic architecture evolution
